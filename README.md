## Descrição

Repositório de governança documental do ecossistema DTD/SETIS/SES-DF.
Contém transcrições estruturadas em Markdown de legislações, portarias,
resoluções e referências internacionais relevantes para a transformação
digital em saúde no Distrito Federal.

Todos os documentos seguem o padrão oficial de transcrição documental
definido pela skill `skill-transcricao-documental` (S05), garantindo
fidelidade, rastreabilidade e reprodutibilidade.
